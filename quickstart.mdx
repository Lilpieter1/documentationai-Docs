## Prerequisites

<Callout kind="info" title="Before You Begin">
Ensure you have:
- A Shopify store with admin access
- Store owner or admin permissions
- No credit card required for the free trial
</Callout>

## Create a Free Trial Account

Sign up for Nousu in minutes to access your AI support automation dashboard.

<Steps>
  <Step title="Visit Signup Page" icon="external-link">
    Go to the [Nousu signup page](https://nousu.nl/signup).
  </Step>
  <Step title="Enter Store Details" icon="shop">
    Provide your Shopify store URL and email address.
  </Step>
  <Step title="Verify Email" icon="mail">
    Check your inbox and click the verification link.
  </Step>
</Steps>

## Install the Shopify App

Install the official Nousu app directly from the Shopify App Store.

<Steps>
  <Step title="Search for Nousu" icon="search">
    In your Shopify admin, navigate to **Apps** > **Shopify App Store** and search for "Nousu".
  </Step>
  <Step title="Add App" icon="download">
    Click **Add app** and authorize the required permissions for orders, customers, and refunds.
  </Step>
  <Step title="Connect to Nousu" icon="link">
    Return to your Nousu dashboard and click **Connect Shopify Store**.
  </Step>
</Steps>

## Configure Basic AI Settings

Customize Nousu's responses for your store using the dashboard.

<Tabs default-tab="2">
  <Tab title="Auto-Reply Settings" icon="message-circle">
    Enable automatic responses for common queries like order status.
    
    ```json
    {
      "enabled": true,
      "triggers": ["where is my order", "wismo", "tracking"],
      "template": "Your order #{order_id} is on its way. Track it here: {tracking_url}"
    }
    ```
  </Tab>
  <Tab title="Product Recommendations" icon="shopping-bag">
    Set up upsell prompts based on customer queries.
    
    ```json
    {
      "enabled": true,
      "keywords": ["cold", "warm", "bedding"],
      "products": ["warm-dekbed", "fleece-blanket"]
    }
    ```
  </Tab>
  <Tab title="Escalation Rules" icon="alert-triangle">
    Define when to forward to human agents.
    
    ```json
    {
      "threshold": 3,
      "forward_to": "support@yourstore.com",
      "include_context": true
    }
    ```
  </Tab>
</Tabs>
