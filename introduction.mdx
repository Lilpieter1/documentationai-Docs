## Overzicht

Nousu transformeert jouw Shopify-klantenservice door repetitieve vragen zoals orderstatuscontroles en retourverwerking te automatiseren. Je zet een AI-collega in die 80% van de routine-tickets afhandelt, waardoor je team tijd overhoudt voor complexe escalaties en verkoopkansen. ## Snelle Start

Krijg Nousu in vier stappen werkend op je Shopify-winkel.

<Steps>
  <Step title="Aanmelden" icon="user-plus">
    Maak je account aan op `https://nousu.nl/signup`. Geen creditcard nodig.
  </Step>
  <Step title="App Installeren" icon="download">
    Voeg de Nousu-app toe vanuit de Shopify App Store.
    
    <CodeGroup tabs="Shopify CLI,npx">
      ```bash
      shopify app install nousu-support
      ```
      ```bash
      npx @shopify/create-app@latest nousu-integration
      ```
    </CodeGroup>
  </Step>
  <Step title="AI Configureren" icon="settings">
    Verbind je winkel en train Nousu op je producten via het dashboard op `https://dashboard.example.com`.
  </Step>
  <Step title="Lancering" icon="rocket">
    Schakel live chat in. Monitor prestaties in real-time.
  </Step>
</Steps>
